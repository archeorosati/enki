
## Report delle Operazioni del 29/07/2024

### Descrizione delle Attività

Oggi, è stata eseguita un'operazione per riorganizzare le immagini all'interno del dataset "DesertPASTPilot_dataset1". L'obiettivo era estrarre un elenco di immagini da cartelle già organizzate (con fonte Google Satellite) e utilizzarlo per identificare e spostare immagini con lo stesso nome dalla cartella non organizzata (stesse immagini ma prese da Bing Satellite) a specifiche cartelle di destinazione. Il fine è quello di aumentare il numero di esempi dei medesimi siti in questo caso raddoppiandoli, per migliorare l'allenamento dell'algoritmo.

### Workflow Dettagliato

1. **Estrazione dell'elenco delle immagini dalla cartella organizzata**
   - Cartella di riferimento: /Users/paolorosati/Library/CloudStorage/GoogleDrive-paolo.rosati@uniroma1.it/Il mio Drive/DesertPASTPilot_dataset1/Archaeological sites/Tell/
   - Comando utilizzato:
     ```
     ls > elenco_immagini.txt
     ```
   - Questo comando ha creato un file `elenco_immagini.txt` contenente i nomi di tutti i file presenti nella cartella organizzata.

2. **Copia delle Immagini da Cartella Non Organizzata**
   - Cartella non organizzata: /Users/paolorosati/Library/CloudStorage/GoogleDrive-paolo.rosati@uniroma1.it/Il mio Drive/DesertPASTPilot_dataset1/B_archaeological_sites_tot
   - Cartella di destinazione principale: /Users/paolorosati/Library/CloudStorage/GoogleDrive-paolo.rosati@uniroma1.it/Il mio Drive/DesertPASTPilot_dataset1/Archaeological_sites_1/Tell/
   - Sottodirectory di destinazione (come le originali):
     - Tell_in_artificial_agricoltural_areas/
     - Tell_in_bare_abandoned_areas/
     - Tell_in_construction_areas/
     - Tell_in_continuos_urban_areas/
     - Tell_in_desert_sands_rocks/
     - Tell_in_discontinuos_urban_areas/
     - Tell_in_natural_arable_areas/
     - Tell_in_permanent_crops/
     - Tell_in_water/

   - Script utilizzato per la copia delle immagini:
     ```
     #!/bin/bash
     organizzata="/Users/paolorosati/Library/CloudStorage/GoogleDrive-paolo.rosati@uniroma1.it/Il mio Drive/DesertPASTPilot_dataset1/Archaeological sites/Tell/"
     non_organizzata="/Users/paolorosati/Library/CloudStorage/GoogleDrive-paolo.rosati@uniroma1.it/Il mio Drive/DesertPASTPilot_dataset1/B_archaeological_sites_tot"
     destinazione="/Users/paolorosati/Library/CloudStorage/GoogleDrive-paolo.rosati@uniroma1.it/Il mio Drive/DesertPASTPilot_dataset1/Archaeological_sites_1/Tell/"

     while IFS= read -r file; do
         if [[ -f "$non_organizzata/$file" ]]; then
             cp "$non_organizzata/$file" "$destinazione/"
         fi
     done < "$organizzata/elenco_immagini.txt"
     ```
   - Questo script ha iterato attraverso l'elenco delle immagini estratte, cercando i file corrispondenti nella cartella non organizzata e copiandoli nelle rispettive sottodirectory della cartella di destinazione.

### Risultati

Le operazioni sono state completate con successo. Le immagini dalle sottodirectory della cartella organizzata sono state correttamente identificate e copiate nelle sottodirectory corrispondenti all'interno della nuova cartella di destinazione. La struttura delle sottocartelle è stata mantenuta per garantire una facile identificazione e accesso alle immagini.
